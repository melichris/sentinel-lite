# Sentinel Lite

Sentinel Lite is a lightweight security dashboard that simulates monitoring of user access and system activity.

## 🚀 Features
- Authentication (JWT-based)
- Protected routes
- Dashboard interface
- API integration

## 🛠 Tech Stack
- Vue 3
- Vue Router
- Axios
- Node.js (backend)
